# Запуск проекта

Для непрерывной сборки проекта без постоянного запуска:

```bash
    gradle build -continuous
```

Для запуска сервиса:

```bash
    gradle bootRun
```

Проверяем работоспособность:

```bash
    open http://localhost:8080/hello
```

Запуск тестов:

```bash
    gradle test
```

Swagger UI:

```bash
http://localhost:8080/swagger-ui/index.html
```
